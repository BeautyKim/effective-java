# ordinal 메서드 대신 인스턴스 필드를 사용하라

- ordinal 메서드는, 열거 타입 상수가, 해당 열거 타입에서 몇번째에 위치하고 있는 지를 반환하는 메서드 이다.
- 열거타입의 위치가 바뀌면?
  - ordinal method를 통해 반환되는 값도 바뀐다.
  - 개발자가 미처 예상치 못한 문제가 발생할 수 있음 --> 사용하지 말자
- 그 대신 **인스턴스** 필드에 저장하도록 하자
